# 🗺️ MovieSwipe - Roadmap de Desarrollo

## Visión
App de recomendación de películas estilo Tinder con algoritmo inteligente de aprendizaje.

---

## 📅 FASE 1: MVP (Junio 2024) ✅

### Características
- [x] 6 géneros principales
- [x] 3 filtros de ordenamiento (Tendencias, Popularidad, Estrenos)
- [x] Swipe básico con botones
- [x] Algoritmo: mejor rating de las aceptadas
- [x] UI/UX profesional estilo Netflix
- [x] Despliegue en Netlify (PWA)

### Objetivo
Validar idea con 500-1,000 usuarios activos

---

## 📅 FASE 2: Feedback y Monetización (Mes 3-4)

### Características Gratis
- [ ] Formulario de feedback post-veredicto
- [ ] Guardar preferencias en localStorage
- [ ] Estadísticas básicas del usuario

### Características Premium ($2.99/mes)
- [ ] Filtros avanzados:
  - Por actor/director
  - Por década/año específico
  - Por premios (Oscars, Golden Globes)
  - Por plataforma de streaming
  - Por duración
- [ ] Modo "Descubrimiento" (películas poco conocidas)
- [ ] Listas personalizadas ilimitadas
- [ ] Sin límites de swipes diarios

### Objetivo
Conseguir 50 primeros suscriptores de pago

---

## 📅 FASE 3: IA Personalizada (Mes 6+)

### Backend
- [ ] Integración con Supabase/Firebase
- [ ] Base de datos de usuarios y preferencias
- [ ] Sistema de autenticación

### Algoritmo de IA
- [ ] Análisis de patrones de aceptación/rechazo
- [ ] Recomendaciones basadas en historial
- [ ] "Modo gusto abierto": slider de personalización
  - Más conocido ↔️ Más desconocido
  - Más reciente ↔️ Más clásico
  - Más mainstream ↔️ Más indie
- [ ] Aprendizaje continuo por usuario

### Objetivo
500 suscriptores = ~$1,500/mes

---

## 🎯 Métricas de Éxito

### Corto plazo (Mes 1-2)
- 1,000 usuarios registrados
- 300 usuarios activos semanales
- 50% tasa de retención D7

### Medio plazo (Mes 3-4)
- 5% conversión a premium
- 50 suscriptores de pago
- $150 MRR (Monthly Recurring Revenue)

### Largo plazo (Mes 6+)
- 10,000 usuarios totales
- 500 suscriptores premium
- $1,500 MRR
- 4.5+ estrellas en reseñas

## 📊 Tech Stack Actual

- **Frontend:** HTML5, CSS3, JavaScript Vanilla
- **API:** TMDB (The Movie Database)
- **Hosting:** Netlify (gratis)
- **Control de versiones:** Git + GitHub
- **PWA:** Manifest + Service Worker

## 🔮 Tech Stack Futuro (Fase 3)

- **Backend:** Supabase o Firebase
- **IA:** TensorFlow.js o Python + FastAPI
- **Analytics:** Google Analytics / Plausible
- **Pagos:** Stripe o Lemon Squeezy
